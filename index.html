Quintessential Tooltip

Pretty basic tooltip if you ask me, just add options object (or not) and you are good to go.
Setup of tooltip can be, by position:

Above and Center (Default Tooltip Setup)

$('.target').quintessentialTooltip(); # # #

Above and Left (left alignment)

$('.target').quintessentialTooltip({ above: true, left: true }); # # #

Above and Right (right alignment)

$('.target').quintessentialTooltip({ above: true, right: true }); # # #

Beyond and left (left alignment)

$('.target').quintessentialTooltip({ beyond: true, left: true }); # # #

Beyond and right (right alignment)

$('.target').quintessentialTooltip({ beyond: true, right: true }); # # #

Beyond and center

$('.target').quintessentialTooltip({ beyond: true, center: true }); # # #
